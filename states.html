Favorite State Website

Iowa has not a very slick and clean design but also a very clean layout. Iowa has a couple of quick links avaliable to its visitors. As well the website features icons aswell as a at a glace section such as the Governor and LT. Governor. Iowa also has a very nice and bright color section that helps the background color(white) mesh all together.

Second Favorite Website

Tennesse had a very slick website, and almost took first place. The website featured a nav bar and a search bar. Though I didn't like the a couple of things about the website. For starters the color scheme seemed weird and darker. As well the Nav bar and most of the page had a horrible text. This doesn't change the fact the page is super easy to navigate. Tennesse page had no quick links for useful information but had a great navigation area.

Third Favorite State Website

While Oklahoma have a very pretty website, I didn't think it earned the number one title or even second spot. I struggled to find any navigation on the website only. I could search and the page lacked a navigation bar. I did like the images provided, aswell the page had some very key resource quickly avaliable. The webpage was mobile friendly and had a very pretty color scheme.

Honorable Mention

While vermonts page scaled quite easily for mobile devices the page was quite confusing. The page had a drop down navigation bar, and was very easy to navigate. Though its color theme and use of so many images made the page confusing. The information was spread across mutltiple pages but on the front page, which was quite confusing.

Non-Honorable mention....

Alaska's State Website for starters is not mobile friendly. The webpage looks out dated and really need work. While the page does feature a nav bar, the color scheme and design layout with the rounded corners was quite embrassing to even have on my laptop. Though the page did have some fun aspects, such as the flag status. The page just didn;t seem to flow, from the weird social media tags, and the blue to red button highlighting.
